# Smart Task Manager 🎯

Your All-in-One Productivity Solution
Built with passion by Maheen Afan

## ✨ Key Features
* Add Tasks – Create tasks with custom categories and set priorities.
* View Tasks – Display all your tasks in a clean, formatted view.
* Mark Complete – Easily toggle task status between Pending and Completed.
* Delete Tasks – Remove tasks you no longer need.
* Task Statistics – Track your progress with a visual progress bar.
* Persistent Storage – Uses SQLite database for secure storage.
* Error Handling – Robust validation for empty entries.
* User-Friendly UI – Intuitive menus with emoji indicators.
* Motivation Section – Get daily motivation quotes.
